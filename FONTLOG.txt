FONTLOG for the Amika fonts

This file provides detailed information on the Amika Font Software.

This information should be distributed along with the Amika fonts and any
derivative works.

Basic Font Information

Amika fonts are a set of clean, utilitarian fonts, supporting Latin and Devanagari scripts.
They are tailored for body text on the web and low resolution cellphones.

To contribute to the project contact Pablo Impallari at impallari@gmail.com
